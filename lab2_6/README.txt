Лабораторная работа No7. Бинарные деревья.

Выполнил: Иванюк Павел Игоревич

Задание:
1. Написать программу для создания произвольного бинарного дерева.
2. Произвести удаление элементов дерева, выбираемых пользователем.
3. Выполнить задание по вариантам.
4. Добавить визуализацию полученного дерева.

Задание варианта:
Пусть имеется бинарное дерево T. Сформировать два идеально
сбалансированных дерева из отрицательных и неотрицательных элементов дерева T.
