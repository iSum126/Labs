Лабораторная работа No6. ОЧЕРЕДЬ.

Выполнил: Иванюк Павел Игоревич

Вариант 7

Задание 1
Создать очередь для символов. Максимальный размер очереди вводится с экрана. Создать
функции для ввода, вывода и определения размера очереди. Ввести эталонный символ. Вводить
символы с экрана в очередь до встречи эталонного. Вывести размер очереди.


Задание 2
Создать дек для символов. Максимальный размер дека вводится с экрана. Создать функции
для ввода, вывода и определения размера дека. Ввести эталонный символ. Вводить символы в дек
поочередно справа и слева до встречи эталонного. Вывести размер дека.


Задание 3
Введите строку. Проанализировав в программе содержимое строки, выберите из нее числа и
занесите в очередь. Выведите содержимое очереди на экран и посчитайте сумму этих чисел. Решение в
программе оформляйте через подпрограммы.
